Remarks

If the command is included in my ASE_Assignment.AppCommandFactory, for example the "drawto" command, the parser will return my ASE_Assignment.AppDrawTo command instead of the original one. For other command types, the method delegates to the original BOOSE.CommandFactory to create the command.
